API costs vs. chat subscriptions

This tool uses the API, which is separate from your Claude, Gemini, or ChatGPT chat subscriptions. API usage is billed per token (roughly 1 token ≈ 1 word) and requires its own billing setup.

Even if you pay for Claude.ai Pro, Gemini Advanced, or ChatGPT Plus, those subscriptions do not cover API usage. API costs are charged separately.

Pricing guide

API costs are charged per token (roughly 1 token ≈ 1 word). Images also consume tokens.

Claude (Anthropic) — requires a credit card, pay as you go:

  • Haiku 4.5 — cheapest (~$0.001 per 1K output tokens)
  • Sonnet 4.6 — mid-range (~$0.005 per 1K output tokens)
  • Opus 4.6 — most expensive (~$0.015 per 1K output tokens)

Gemini (Google):

  • 2.5 Flash — free tier available, best for everyday use
  • 2.5 Pro — free tier with lower limits
  • 2.0 Flash / 1.5 series — older models, free tier available

ChatGPT (OpenAI) — requires a credit card, pay as you go:

  • GPT-4o mini — cheapest (~$0.0006 per 1K output tokens)
  • GPT-4o — mid-range (~$0.01 per 1K output tokens)
  • GPT-5.4 / GPT-5 series — most capable and most expensive
  • o-series (o1, o3) — reasoning models, significantly more expensive

Image generation (Image Gen tab):

  • GPT Image 1 mini — ~$0.005 per image (cheapest)
  • GPT Image 1 — ~$0.04 per image
  • GPT Image 1.5 — ~$0.04+ per image (best quality)
  • Nano Banana (Google) — varies by model; free tier may apply
The token counts shown in each response (e.g. 30↑ 309↓) show how many tokens were sent up (↑) and received down (↓) for that message. Multiply by the model's price per token to estimate cost.

Understanding the controls & their cost impact

The toggle pills above the input box let you control which AIs respond and how.

Claude toggle (orange) — enables or disables Claude. When off, Claude's card is hidden.

Gemini toggle (blue) — enables or disables Gemini. When off, Gemini's card is hidden.

ChatGPT toggle (green) — enables or disables ChatGPT. When off, ChatGPT's card is hidden.

When only one AI is enabled, its response takes the full width. When two are enabled, they split the screen. All three enabled shows three columns.

Single-turn mode (yellow when on) — when on, each message is sent as a standalone question with no conversation history. The AI won't remember earlier turns. When off (default), the full history is sent so the AI can follow along.

Stacked layout (green when on) — when on, responses appear one above the other instead of side by side.

Mode combinations explained

All AIs on · Single-turn OFF (default)
Claude, Gemini, and ChatGPT all respond. Each follow-up sends the full conversation history to all three. Best for comparing reasoning across a real back-and-forth — but the most expensive mode.

All AIs on · Single-turn ON
All three respond, but every message is fresh with no history. Good for quickly comparing isolated answers at lower cost.

One AI on · Single-turn OFF
Only the enabled AI responds and remembers the full conversation. Like using Claude.ai, Gemini, or ChatGPT directly.

One AI on · Single-turn ON
Only one AI, no history. The lowest-cost mode for quick independent answers.

Switching Single-turn mode on or off automatically clears the current conversation history to avoid mixing context between modes.

Images add cost

When you attach an image in the Query tab, it is encoded and sent to the API along with your text. Images consume additional tokens — a typical photo can add hundreds to thousands of tokens per message. If you use multi-turn mode with images, those image tokens are re-sent with every follow-up in that conversation.

To keep image costs low, use Single-turn mode for image questions, or start a new chat after getting your answer.

Multi-turn conversations cost more

Every time you send a follow-up message (with Single-turn OFF), the entire conversation history is sent to the API again so the AI has context. A 10-message conversation costs significantly more than 10 separate single questions.

To keep costs low: start a + New chat for each unrelated question, or turn on Single-turn mode. Only use multi-turn when you genuinely need the AI to remember what was said earlier.

Your API keys are sensitive

Treat your API keys like passwords. Anyone who has your key can make API calls charged to your account.

  • Keys are saved locally in your browser — never sent anywhere except the respective AI APIs.
  • Do not share your keys or send this tool to others with your keys already entered.
  • Revoke and regenerate keys at any time: console.anthropic.com (Claude), aistudio.google.com (Gemini), or platform.openai.com/api-keys (ChatGPT).

Troubleshooting

  • "invalid x-api-key" — Claude key is wrong. Get a new one at console.anthropic.com.
  • "API key not valid" — Gemini key is wrong. Get a new one at aistudio.google.com/app/apikey.
  • "Incorrect API key" — ChatGPT key is wrong. Get a new one at platform.openai.com/api-keys.
  • "model not found" — Change the model in the API Keys dropdown.
  • Only one or two responses showing — One or more AIs are toggled off. Click their pill above the input to re-enable.
  • Page won't open — Check the taskbar for the server window. If missing, relaunch Launch Multi-AI.bat, then go to localhost:8731/multi-ai.html manually.
